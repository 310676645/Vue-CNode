# vue-cnode使用vue2.0开发的cnode社区

## 使用技术

Vue2.0 + vue-router + vue-resource + es6 + mint-ui + webpack

## 项目运行

``` bash
# 安装依赖
npm install

# 开发预览,执行命令后浏览器自动打开http://localhost:8080
npm run dev

# 生产环境打包
npm run build
```
